# FUTURE_DS_03
# Task 3 – College Event Feedback Analysis

This repository contains Task 3 of the Data Science & Analytics Internship at Future Interns.

## Objective
Analyze student feedback collected after college events to understand satisfaction levels,
identify strengths and improvement areas, and provide actionable recommendations
for future event planning.

## Dataset
- Student feedback dataset (rating-based responses)
- Each row represents feedback from a participant
- Ratings provided on a numerical scale

## Analysis Performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Rating distribution analysis
- Correlation analysis between feedback aspects
- Overall satisfaction scoring
- Sentiment classification (Positive / Neutral / Negative)
- Insight generation and recommendations

## Tools & Technologies
- Python
- Jupyter Notebook
- pandas
- matplotlib
- seaborn

## Repository Structure
- `data/` – Dataset used for analysis  
- `notebook/` – Jupyter Notebook with full analysis  
- `visuals/` – Saved charts and graphs  
- `report/` – Final analysis report (PDF)

## Outcome
The analysis provides data-driven insights and practical recommendations
to improve the quality and effectiveness of future college events.
